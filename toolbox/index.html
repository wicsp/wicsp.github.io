Toolbox

这是一个我折腾过的好玩的软硬件工具箱， 你可以在这里找到我使用的工具和软件。

MacOS Apps

Windows Manager

  • AeroSpace Mac 上的窗口管理工具，最好的 i3WM 替代品。
  • SketchyBar 一个在 MacOS 上的状态栏插件，可以自定义状态栏，配合 AeroSpace 使用。

System Customization

  • [HammerSpoon] 一个自动化工具，可以自定义各种快捷键， 并运行脚本。
  • [Karabiner-Elements] 键盘映射工具。

Basic Tools

  • kitty 终端工具，支持 GPU 加速。
  • [Bob] 翻译工具，感觉现在挺多替代品了。
  • [InputSourcePro] 在不同应用之间自动切换输入法。
  • Raycast 一个快速启动工具，类似 Alfred，代替系统默认 Spotlight。
  • Snipaste 截图工具，支持截图、标注、贴图等功能。
  • Ice Menu Bar 管理工具。 ### Softwares
  • Obsidian 一个知识管理工具，支持 Markdown。
  • Dash 一个文档查询工具，支持各种语言的文档查询。
  • RunCat 一个可爱的跑步猫，可以显示 CPU 占用率。

命令行工具

Shell

  • Zsh 目前使用 powerlevel10k 主题。

Tools

  • Lazydocker Docker 管理工具
  • Lazygit Git 管理工具
  • Tmux 终端复用工具
  • Atuin 使用 Sqlite 管理历史命令，贼好用。
  • thef**k 一个自动纠正命令的工具， 用于纠正命令拼写错误。为了避免 Copilot 罢工，我用了两个星号代替了其中的两个字母。

Coding

IDE

  • VSCode 主力 IDE
  • NVIM 作为 VSCode 的补充，主要用于快速编辑和查看文件，配好了可以装逼。

IDE 插件

VSCode

  • Copilot 代码提示、补全和生成， 主流 IDE 都支持。
  • ErrorLens VSCode 中高亮代码中的错误和警告。(JetBrains IDE 有类似插件)
  • Code Ace Jumper 一个快速跳转到任意位置的插件。
  • Meterial Theme 一个主题插件。
  • Material Icon Theme 为文件图标添加颜色。
  • Path AutoComplete 自动补全文件路径。
  • Rainbow CSV 为 CSV 文件添加颜色。
  • VSCode NeoVim 为 VSCode 添加 NeoVim 支持。
  • SQLite 为 VSCode 添加 SQLite 支持。
  • SQLite Viewer SQLite 查看器。
  • Todo Tree TODO 查看和管理，配合 Maple 字体使用。

Blog

静态博客（网站）框架

  • Hexo
  • Hugo (感觉主题都有点简陋？)

静态博客主题

  • NexT
  • Fluid (之前太久没更新，重启博客的时候就放弃了，最近好像又更新了)

Markdown 编辑器

  • VSCode （VSCode 持续发力中，但是基本不支持所见即所得，不如 Typora）

博客发布

  • GitPage Github 官方静态网站托管服务。

Python Packages

  • Ruff, 用于替代 pylint 的代码检查工具。
  • pytest, 用于替代 unittest 的单元测试工具。
  • fastapi, 用于替代 flask 的 web 框架。
